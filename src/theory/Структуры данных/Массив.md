## Статический массив
```
    1. Все элементы одного типа
    2. Фиксированное число элементов
    3. Значения элементов менять можно
    4. Все элементы хранятся в памяти рядом друг с другом без пропусков 
    
    Достоинства:
    1. Чтение и запись O(1), т.е. "мгновенный" доступ к любому элементу
    2. Прост в реализации
    3. Удобен для небольших наборов данных
    
    Недостатки:
    1. Вставка и удаление O(n)
    2. Хранение данных в непрерывной области памяти. Не эффективно для больших наборов данных
    3. Нельзя изменить количество элементов в процессе раоты программы
    
    В Python нет статического массива. Статические массивы есть в C++.
```

## Динамический массив
```
      Динамический массив - это массив, который может менять число своих элементов в процессе
    работы программы.
    
      Динамические массивы реализуются на основе статических массивов и хранят данные в
    непрерывной области памяти.
    
      Если начального размера динамического массива недостаточно, то создается новый массив
    в несколько раз больше предыдущего с копированием всех прежних значений. 
    
      В Python list реализован как динамический массив. list хранит данные только одного типа, а
    именно list хранит ссылки на объекты (С. Балакирев).
```
